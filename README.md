### 🔧 What I Build

**[netty-rpc](https://github.com/xuweilin2014/netty-rpc)** — A lightweight RPC framework built from scratch with Netty + Zookeeper + Spring.  
20+ production-grade features including service discovery, 3 load balancing strategies, 4 cluster fault-tolerance modes, rate limiting, SPI, graceful shutdown, and more. Architecturally inspired by Dubbo — built to deepen my understanding of it.

**[zeromq](https://github.com/xuweilin2014/zeromq)** — A minimal message queue system built on Netty, supporting multi-producer/consumer topologies, keyword routing, consumer clustering, and async batched delivery.

**[java-nio-server](https://github.com/xuweilin2014/java-nio-server)** — A Java NIO server implementation for learning low-level network programming.
